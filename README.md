# 🚀 Desarrollo Java: De Escritorio a Microservicios

¡Bienvenido a mi repositorio de aprendizaje y proyectos en **Java**! Este espacio está diseñado para recopilar código, prácticas y proyectos que abarcan todo el ecosistema de Java, desde aplicaciones de escritorio tradicionales hasta arquitecturas modernas orientadas a microservicios en la nube.

---

## 🛠️ Tecnologías y Tecnologías Cubiertas

El repositorio está organizado según los diferentes módulos y paradigmas del ecosistema Java:

### ☕ Java Core & Desktop
* **Java Desktop:** Creación de interfaces gráficas de usuario (GUI) utilizando Swing / JavaFX.
* **JUnit:** Pruebas unitarias y de integración para asegurar la robustez y calidad del código.

### 🌐 Desarrollo Web & Java EE (Jakarta EE)
* **Java Web / Java EE:** Manejo de Servlets, JSPs y despliegue en contenedores de servlets (como Apache Tomcat).
* **Full Stack:** Integración de backend en Java con interfaces de usuario dinámicas.

### 🗄️ Persistencia de Datos (ORM)
* **JPA (Java Persistence API):** Estándar de persistencia de objetos en Java.
* **Hibernate:** Implementación de ORM para mapear objetos a bases de datos relacionales de forma eficiente.

### 🍃 Ecosistema Spring
* **Spring Boot:** Desarrollo ágil de aplicaciones e APIs REST listas para producción de forma simplificada.
* **Spring Cloud:** Herramientas para la configuración y gestión de sistemas distribuidos.
* **Microservicios:** Arquitectura desacoplada, patrones de diseño de microservicios, service discovery (Eureka), API Gateway y comunicación inter-servicio.
